- [Helix](https://helix-editor.com/)
- [Ghostty](https://ghostty.org/)

## Install

```sh
git clone https://github.com/your-user/yerba-theme
cd yerba-theme

# install for a specific app
./install.sh helix
./install.sh ghostty

# or install all
./install.sh all
```
